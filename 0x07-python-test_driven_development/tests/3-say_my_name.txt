===========================
How to use 3-say_my_name.py
===========================

The function ``say_my_name()`` will display "My name is" followed by the first
and optional last name supplied to it

    	>>> say_my_name = __import__("3-say_my_name").say_my_name

Strings
=======
The first and last name inputs must be strings

    	 >>> say_my_name("Derek", "Kwok")
	 My name is Derek Kwok

	 >>> say_my_name("John", "Smith")
	 My name is John Smith

	 >>> say_my_name("Walter", "White")
	 My name is Walter White

	 >>> say_my_name("Bob")
	 My name is Bob 

	 >>> say_my_name("Derek")
	 My name is Derek 

	 >>> say_my_name("D", "K")
	 My name is D K

	 >>> say_my_name("    ", "    ")
	 My name is          

	 >>> say_my_name(" ", " ")
	 My name is    

	 >>> say_my_name(" ", "Kwok")
	 My name is   Kwok

	 >>> say_my_name("Derek", " ")
	 My name is Derek  

	 >>> say_my_name("", "")
	 My name is  

	 >>> say_my_name("Derek", "")
	 My name is Derek 

	 >>> say_my_name("", "Kwok")
	 My name is  Kwok
